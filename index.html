Master Thesis

My master's thesis focuses on improving accuracy in insect-pest recognition algorithms using the largest pest-insect dataset, the IP102. To this end, I implemented data augmented EfficientNet models with TensorFlow, which use a clustering algorithm to address class imbalance and intra-class variance. In the end, I achieved a 3.7% average improvement in average classification. For future research on this dataset, I recommend adding a triplet loss function to better detect subtle image features address and address inter-class similarity.
